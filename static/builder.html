Ollama = herramientas locales y /claude. Swarm puede usar también OpenRouter/Groq (claves abajo). El chat Puter usa Puter.js (CDN) configurado abajo.

          
Puter.js (chat en el navegador, sin API key en el servidor): la librería oficial se carga desde https://js.puter.com/…. Documentación: developer.puter.com · docs.puter.com. Tras cambiar la URL, recarga la página (F5). En Node se usa init(process.env.puterAuthToken); en este dashboard el login es por popup en el navegador.

          
          
          
Groq: console.groq.com o GROQ_API_KEY. OpenRouter: solo modelos gratuitos (:free o precio 0); el servidor rechaza modelos de pago.
